Color Palettes Paulina Gary

We know our niche and the broad musical types and how they might influence your color choice. But, do not panic, we have you covered.

Here are several color palettes for you to chose, and the code you'll need to use in order to use each background-color. Every palette also consideres black and white.

Basic colors

  • Black:
    rgb(0, 0, 0)

    Example code:

                                 <button class="c1"> Send </button>
  • White:
    rgb(255, 255, 255)

    Example code:

                                 <button class="c2"> Send </button>

Lists Paulina

Unordered Lists

Our unordered list uses disc list style, and it would look like this.

  • Item 1
    • Item 1.1
    • Item 1.2
    • Item 1.2
  • Item 2
  • Item 3

Example code:

                    
                        <ul class="disc1">
                            <li>Item 1</li>
                                <ul class="disc2">
                                    <li> Item 1.1</li>
                                    <li> Item 1.2</li>
                                    <li> Item 1.2</li>
                                </ul>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ul>

Ordered Lists Paulina

Our ordered list uses decimal list style, and it would look like this.

  1. Item 1
    1. Item 1.1
    2. Item 1.2
    3. Item 1.2
  2. Item 2
  3. Item 3

Example code:

                    
                        <ol class="decimal">
                            <li>Item 1</li>
                                <ol class="decimal2">
                                    <li> Item 1.1</li>
                                    <li> Item 1.2</li>
                                    <li> Item 1.2</li>
                                </ol>
                            <li>Item 2</li>
                            <li>Item 3</li>
